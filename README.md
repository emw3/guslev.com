# My Website

## Requirements

- Node JS (v14.17)
- NPM (v6.14)

## Installation

```bash
npm i
```

## Build

```zsh
npm run build
```

## Run

```bash
npm run start
```

## Development

```bash
npm run dev
```

## Test

```bash
npm run cypress:headless
```
Don't forget to run the Build and Run commands before testing.
